CHATBOT


Our teams chatbot is designed to assist Melbourne commuters with public transport
planning, providing an intuitive chat interface for querying route, schedule, and accessibility
information for trains, trams,and buses. It integrates real-time data and mapping features,
making it a versatile tool for navigating the city. It is built primarily using the RASA
conversational platform, using GTFS data supplied from Public Transport Victoria.

Key Features

Route Planning and Scheduling:

  • The chatbot helps users find the best routes, including options with minimal stops or the quickest travel times.
  • It provides real-time schedule updates for trains, trams, and buses, ensuring users have up-to-date information.

Interactive Mapping:

  • The bot generates maps for public transport stops, allowing users to view tram, train, or bus station locations.
  • Directions from user-provided locations to destinations are supported with custom route maps.

Accessibility Support:

  • Users can inquire about station accessibility features, such as lifts or parking, or find stations with specific facilities.
