About the Physical Activity Dashboard

This dashboard is designed to provide insights into physical activity trends among males and females in Melbourne. Users can visualize and explore data to better understand how different demographics participate in physical activities.

The dashboard features:

  • Interactive visualizations with filters for activity type and responses.
  • Dark and light mode for better user experience.
  • Error handling and user-friendly messages.
  • Future enhancements like downloadable graphs and summaries.
