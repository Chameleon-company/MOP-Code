Our Project

Data 2 Intelligence (D2I) Melbourne City is a capstone project for Deakin Masters students. It allows Deakin students to work with industry partners from Melbourne City and find innovate uses for City of Melbourneâ€™s Open data.

Our goal is to use open data to track predict metrics that will assist the City of Melbourne achieve its environment care goals, more information about our project can be found here

  • Dashboards

    The team has developed some dashboards that give insights into pedestrian data, these are:

    Historical Pedestrian Data Metrics

    Pedestrian data enriched with Microclimate Data

    Pedestrian data analysis by Sensor

    Pedestrian data analysis by Postcode

  • Heat Map

    Our heat map shows where there are the most pedestrians at one time. Our heat map page features a video loop showing pedestrian traffic in Melbourne over time and the ability to generate a pedestrian heat map for a specific date and time.

  • Pedestrian Prediction

    Our team created an algorithm that predicts total daily pedestrian count using a number of features including weather, COVID restriction stage and public holidays.
