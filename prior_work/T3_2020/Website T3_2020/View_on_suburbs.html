A View on Suburbs in the Melbourne CBD

This dashboard is designed to provide pedestrian traffic count on bases of postcodes in CBD area. It provides a magnified view of pedestrian traffic according to a particular postcode in CBD area. It is using pedestrian traffic sensor locations and post code shape file. Each sensor is mapped to a postcode according to its location. Data is collected from three different sources which includes pedestrian traffic data, sensor locations and post code shape file.
