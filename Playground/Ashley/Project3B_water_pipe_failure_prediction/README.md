## Project Description
Predictive system that identifies water pipes most likely to fail based on age, material type, soil conditions, and repair history.

## Project Objectives
Develop a predictive model for pipe failure risk Analyse real-world infrastructure datasets Identify high-risk pipe segments Provide data-driven maintenance recommendations

## Datasets Used
EPA Water Infrastructure Data - data.epa.gov (5,001 records)
Syracuse NY Water Main Breaks - data.syrgov.net (1,000 records)
Key Technologies
XGBoost - Primary predictive model Random Forest - Ensemble benchmark model Gradient Boosting - Additional comparison model SMOTE - Class imbalance handling pandas/NumPy - Data processing matplotlib/seaborn - Visualisation scikit-learn - Model evaluation and preprocessing
