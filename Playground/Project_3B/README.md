# Water Pipe Failure Prediction System

## Project 3B - Company Chameleon

### Team Members
- Rupanshi
- Ashley Mathew

### Project Description
Predictive system that identifies water pipes most likely to fail
based on age, material type, soil conditions, and repair history.

### Project Objectives
Develop a predictive model for pipe failure risk
Analyse real-world infrastructure datasets
Identify high-risk pipe segments
Provide data-driven maintenance recommendations

### Datasets Used
1. EPA Water Infrastructure Data - data.epa.gov (5,001 records)
2. Syracuse NY Water Main Breaks - data.syrgov.net (1,000 records)

### Key Technologies
XGBoost - Primary predictive model
Random Forest - Ensemble benchmark model
Gradient Boosting - Additional comparison model
SMOTE - Class imbalance handling
pandas/NumPy - Data processing
matplotlib/seaborn - Visualisation
scikit-learn - Model evaluation and preprocessing
