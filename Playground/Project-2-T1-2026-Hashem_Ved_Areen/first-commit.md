Project 2: Smart Street Lighting Design System
Project Description
This project develops an AI system that proposes optimal streetlight placement for urban areas such as
parks, pathways, and streets. The system considers factors such as pedestrian traffic, safety, lighting
coverage, and energy efficiency.
The goal is to assist city planners in designing lighting systems that improve safety while minimizing
energy cost.
Gap Addressed
Traditional street lighting design often follows fixed rules without considering real urban activity patterns.
An AI-assisted system can analyze data and suggest lighting layouts that better match local conditions.
Project Objectives
• Develop an AI assistant that proposes lighting layouts
• Use urban data such as pedestrian activity and location maps
• Estimate energy consumption and installation cost
• Provide explanations for design recommendations
Optimization Method
Optimization is only required for Distinction (D) and High Distinction (HD) students.
Students can improve system performance using methods such as:
• Prompt engineering for design reasoning
• Retrieval-Augmented Generation using lighting standards
• LLM reasoning strategies for spatial design
• Optimization algorithms such as genetic algorithms
Students are free to explore other optimization techniques.
Public Datasets
• OpenStreetMap geographic data
• City open data portals (Melbourne Open Data)
• Australian Bureau of Statistics population density datasets
• Public crime statistics datasets
Example
A user asks:
"Design lighting for a park with high evening activity."
The system proposes:
Number of lights: 18
Spacing: 25 meters
Recommended technology: LED solar lights
Estimated yearly energy cost: $1,200
This is similar to how Google Maps recommends optimal travel routes.
