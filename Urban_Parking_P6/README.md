# Urban Parking Demand Forecasting using Graph Neural Networks (GNN)

## Project Overview
This project aims to predict parking availability across urban areas using spatiotemporal deep learning. We use Graph Neural Networks (GNN) to capture relationships between nearby parking locations.

---

## Objectives
- Forecast parking demand across different locations and time periods  
- Reduce traffic congestion caused by searching for parking  
- Analyze spatial and temporal parking patterns  

---

## Dataset
- Dataset: Melbourne Parking Dataset  
- Source: (Add link here later)  
- Key Features:
  - Parking Bay ID  
  - Timestamp  
  - Latitude & Longitude  
  - Occupancy Status  

---

## Proposed Approach (GNN)

- **Nodes** → Parking Bays  
- **Edges** → Nearby parking bays (based on distance)  
- **Features** → Occupancy, time (hour/day), location  
- **Target** → Predict future parking availability
